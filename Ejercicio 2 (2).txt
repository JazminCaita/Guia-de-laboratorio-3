% Definimos el rango de x y la cantidad de elementos
x = linspace(-10, 10, 40); % Genera un vector de 40 elementos equiespaciados entre -10 y 10

% Calculamos las funciones para cada valor de x
y1 = abs(x);             % Valor absoluto de x
y2 = sqrt(x);            % Raíz cuadrada de x (se asume que x >= 0)
y3 = cos(x);             % Coseno de x
y4 = exp(x);             % Exponencial de x
y5 = log10(abs(x));      % Logaritmo base 10 del valor absoluto de x (se asume que x != 0)

% Creamos una nueva figura para la gráfica
figure;

% Graficamos cada función
plot(x, y1, 'r-', 'LineWidth', 2);   % Grafica y1 en color rojo, con línea sólida y ancho de 2
hold on;                             % Mantenemos la figura para agregar más gráficos
plot(x, y2, 'g--', 'LineWidth', 2);  % Grafica y2 en color verde, con línea discontinua y ancho de 2
plot(x, y3, 'b-.', 'LineWidth', 2);  % Grafica y3 en color azul, con línea de puntos y guiones y ancho de 2
plot(x, y4, 'm:', 'LineWidth', 2);   % Grafica y4 en color magenta, con línea punteada y ancho de 2
plot(x, y5, 'c-', 'LineWidth', 2);   % Grafica y5 en color cian, con línea sólida y ancho de 2

% Agregamos etiquetas y título
xlabel('x');              % Etiqueta el eje x como "x"
ylabel('y');              % Etiqueta el eje y como "y"
title('Gráficas de funciones'); % Título de la gráfica
legend('|x|', 'sqrt(x)', 'cos(x)', 'exp(x)', 'log10(|x|)', 'Location', 'northwest'); % Leyenda con los nombres de las funciones
grid on;                  % Agrega una cuadrícula a la gráfica
hold off;                 % Finaliza la edición de la gráfica


Este código genera un gráfico con cinco funciones diferentes en el rango de [-10, 10], utilizando exactamente 40 elementos. 
Cada función se calcula para cada valor de x en el rango dado. Luego, se grafican las funciones y se agregan etiquetas a los ejes, 
un título a la gráfica y una leyenda que identifica cada función. Además, se añade una cuadrícula a la gráfica para facilitar la lectura 
de los valores.


% Definimos el rango de x y la cantidad de elementos
x = linspace(-10, 10, 40); % Genera un vector de 40 elementos equiespaciados entre -10 y 10

% Calculamos las funciones para cada valor de x
y1 = abs(x);             % Valor absoluto de x
y2 = sqrt(x);            % Raíz cuadrada de x (se asume que x >= 0)
y3 = cos(x);             % Coseno de x
y4 = exp(x);             % Exponencial de x
y5 = log10(abs(x));      % Logaritmo base 10 del valor absoluto de x (se asume que x != 0)

% Creamos una nueva figura para la gráfica
figure;

% Graficamos cada función

plot(x, y1, 'r', 'LineWidth', 2)   % Grafica y1 en color rojo, con línea sólida y ancho de 2

% Mantenemos la figura para agregar más gráficos
plot(x, y2, 'g--', 'LineWidth', 2)  % Grafica y2 en color verde, con línea discontinua y ancho de 2

plot(x, y3, 'b-.', 'LineWidth', 2)  % Grafica y3 en color azul, con línea de puntos y guiones y ancho de 2

plot(x, y4, 'm:', 'LineWidth', 2)   % Grafica y4 en color magenta, con línea punteada y ancho de 2

plot(x, y5, 'c-', 'LineWidth', 2)   % Grafica y5 en color cyan, con línea sólida y ancho de 2
subplot (y1, y2, y3, y4, y5)
% Agregamos etiquetas y título
xlabel('x')              % Etiqueta el eje x como "x"
ylabel('y')              % Etiqueta el eje y como "y"
title('Gráficas de funciones'); % Título de la gráfica
legend('|x|', 'sqrt(x)', 'cos(x)', 'exp(x)', 'log10(|x|)', 'Location', 'northwest') % Leyenda con los nombres de las funciones
grid on                  % Agrega una cuadrícula a la gráfica
hold off                 % Finaliza la edición de la gráfica
